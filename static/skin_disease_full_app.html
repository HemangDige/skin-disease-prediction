About This Project

This Skin Disease Prediction System is a BE Final Year Project developed to assist in early detection of skin diseases using Artificial Intelligence. The system uses the HAM10000 dataset — a large collection of dermatoscopic images — to train a deep learning model that can classify skin lesions into 7 categories.

The project uses a custom Convolutional Neural Network (CNN) trained from scratch for image classification. Additionally, SVM and Random Forest models were trained and compared. The CNN achieved the best validation accuracy of 73.42% on dermoscopic image analysis.
